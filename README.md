# Port Harcourt GIS Fuel Locator

A lightweight GIS-based web application that helps users locate fuel stations across Port Harcourt using an interactive map.  
The system displays real-time station information including PMS price, address, queue status, number of pumps, and accessibility features.  
Designed as part of a GIS research project, it demonstrates how spatial data, GeoJSON, and client-side mapping technologies can support transparency and efficient fuel purchasing.
